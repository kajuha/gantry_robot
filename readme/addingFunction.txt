해결:
x 호밍 및 위치 관련 설정값은 파라미터(런치)에서 설정하는 것으로 처리

x 호밍서비스 삭제, command.srv에 호밍작업을 추가할 것, 호밍은 Z축 이후에 X,Y는 동시에 동작하도록 할 것
x SI 단위계 사용(m, m/s)
x command.srv는 초기화(호밍), 정지, 에러 해제
